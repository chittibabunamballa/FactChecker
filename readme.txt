-------------------How to run-------------------------------
In order to run the tool,first we've to install the WordNet-2.0.exe.
To run the tool, just double click on autorun.bat .
If you have a proxy server,then edit the autorun_proxy.bat by modifying the proxyhost address and proxy port.And then double click on autorun_proxy.bat .
